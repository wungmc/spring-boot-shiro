macOS X 安装 AtiveMQ（5.15.6）：
1. [下载](http://activemq.apache.org/activemq-5156-release.html) Linux版本
2. 解压
3. 终端进入apache-activemq-5.15.6/bin/macosx 目录
4. 执行命令：./activemq start
5. 访问：http://localhost:8161/admin （用户名和密码都为admin）
